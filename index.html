My class projects

Assassin

Players are assigned targets to assasin. Another class, AssassinManager is responsible for managing a game of Assassin. The class tracks the kill ring, graveyard, and provides methods to perform kills, check game status, and print the current state of the game. It uses circular and simple linked lists to represent the players and their relationship.

Where's Waldo

A class is created to represent Waldo's hiding location in an auditorium. The class uses a 2D array to stimulate the auditorium seats, with Waldo randomly placed. Players take turns searching for Waldo, and the class keeps track of the current player, Waldo's found status, and provides methods to search seats and check if they have been searched before. The class also displays the auditorium state, hiding Waldo's location until he is found.

Lost Puppy

The lost puppy class uses recursive backtracking to help a puppy find its way out of a maze. It reads maze data from a file, stores it in a 2D array, and navigates through the maze by marking its path and backtracking when needed. The class includes methods to validate moves, find the start and exit locations, and display the maze. The main method controls the solving process and displays the final result.
